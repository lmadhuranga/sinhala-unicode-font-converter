# Sinhala Selection Converter (Chrome Extension)

Right-click selected Singlish text to convert it to Sinhala Unicode.

## Load Unpacked
1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click "Load unpacked" and select the `extension` folder.

## Notes
- The extension attempts to replace the selected text in editable fields.
- If replacement is not possible, it copies the converted text to clipboard.
