# Privacy Policy for Sinhala Selection Converter

Effective date: February 22, 2026

Sinhala Selection Converter converts user-entered or user-selected Singlish text to Sinhala Unicode in the browser.

## Data Collection
- We do not collect personal data.
- We do not collect browsing history.
- We do not collect authentication data, financial data, health data, location data, or communications.

## Data Processing
- Conversion runs locally in the browser.
- The extension stores optional converter draft/history locally on the user device using browser storage APIs.
- Stored local draft/history is not transmitted to external servers by this extension.

## Data Sharing
- No personal data is sold.
- No personal data is shared with third parties.

## Remote Code
- The extension does not execute remote code.
- All executable code is bundled with the extension package.
